| 模块 | 功能 | 规模 |

## 三、踩坑与修复（14 坑 · 全部已纳入技能）

| # | 坑 | 根因 | 修复方案 |
|---|-----|------|---------|
| 1 | 品类匹配子串污染 | keys 未排序，`芒`匹配到单字 | keys 按长度降序 + 排除单字 |
| 2 | 文案 placeholder 泄漏 | `getInputVal` 未读空返回默认值 | 直接读 `input.value.trim()` |
| 3 | 文案空洞 | 品类知识未融入 | 76条 FRUIT_SELLING 金句引擎 |
| 4 | genPrompt 模板闭合 | 反引号内分号提前闭合 | 整个 return 统一反引号 |
| 5 | 配图"土" | 暖黄 + 道具堆平面 + 数量失衡 | 白平衡 + 3层纵深 + 4-6件 |
| 6 | 构图污染 | "留白给文案""盒盖半开"不存在 | 全局审查删除非参考元素 |
| 7 | emoji 粒子乱码 | surrogate pair 被拆散 | `Array.from` + 禁止国旗 |
| 8 | 白色雪花不可见 | 白底白雪花 | 节日深色背景 + z-index |
| 9 | 节日日期不联动 | `getTodayFestival` 未读 datePicker | 优先读 datePicker.value |
| 10 | API token 被拦截 | token 明文在命令中 | 写 `/tmp/gh_token` 间接引用 |
| 11 | Gitee Pages 不可用 | 需实名认证 | 改用 GitHub Pages |
| 12 | Vercel 国内被墙 | GFW | 最终方案 GitHub Pages |
| 13 | git push 超时 | 网络慢 | GitHub API 直传（5秒内） |
| 14 | PWA 图标/名称不更新 | SW 缓存旧版永不失效 | SW v2 + activate 清旧缓存 + `?v=2` |
